# InputViewDemo
此空间为常用的输入框 左边为key右边为输入框的value  
使用方法
in your root build.gradle at the end of repositories:

	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}

Step 2. Add the dependency

	dependencies {
	        implementation 'com.github.Smartisan1:InputViewDemo:Tag'
	}
